# Show Commands on NX-OS

## NX-OS Version

titanium-kickstart.7.1.0.ZD.0.363.bin

## N7k

```
show hostname
show cdp neighbor
show cdp neighbor | xml
show ip int brief
show ip route local
show ip route ospf
show bgp ipv4 unicast summary
show ip route bgp
```

### IPv6

```
show ipv6 int brief
show bgp ipv6 unicast summary
show ipv6 route
```
