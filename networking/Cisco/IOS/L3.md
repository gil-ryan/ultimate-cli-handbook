# General Layer 3 Commands and Templates

## Infrastructure

### L3 Port-Channel

```
interface range gi0/1 - 2
```

## Routing Protocols

### Classic EIGRP

```
router eigrp 10
network 0.0.0.0
```

## Distribute List

```
access-list 10 permit 10.10.11.0 0.0.0.255
access-list 10 permit 10.10.12.0 0.0.0.255
router ospf 10
redistribute eigrp 10
distribute-list 10 out eigrp 10
```
